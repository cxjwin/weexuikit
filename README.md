# Weex UIKit
This is the Weex UIKit project. cross platform modular high performance render engine for weex and min app


Weex高性能跨平台渲染引擎，性能媲美Flutter和Native，样式能力元强于Flutter。 

工程简介:

  1、WML 高效跨平台线程库
  
  2、Widget高级组件
  
  3、Gesture 手势识别库 
  
  4、Frame渲染线程调度
  
  5、Platform平台对接层
  
  6、Rendering Flexbox布局和样式实现。
